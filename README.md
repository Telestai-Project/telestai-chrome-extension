# ravencoin-chrome-extension

To build and run this locally you neeed to have node.js installed.

# to build

`npm install`

for local development

```
npm start
```

to build for production

```
npm run build
```

to bundle the extension as ZIP-file run

```
npm run build
npm run buildZipFile
```
